## [0.1.0] — 2026-03-04

### Added

- **Multi-provider support**: Gemini, Groq, Ollama, Azure OpenAI, Anthropic
- **Agent types**: `BasicAgent`, `Agent`, `SmartAgent`, `CopilotAgent`, `MCPAgent`
- **Built-in tools**: File system, web search, git, code execution, PDF, Office docs
- **MCP integration**: Model Context Protocol for dynamic tool discovery
- **Streaming**: Real-time token streaming with async callback support
- **Vision**: Multimodal image understanding across supported models
- **Memory**: Session persistence and simple memory stores
- **Telemetry**: Execution tracing and walkthrough generation
- **Skills**: Modular skill packs for domain-specific tasks
- **Hot reload**: Live code reloading during development
