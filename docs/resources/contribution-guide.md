Thanks for helping improve Logicore.

## Ways to contribute
- Fix or clarify docs
- Improve examples and code samples
- Report bugs with reproducible steps
- Propose features with a concise problem statement

## Workflow
1. Fork and create a branch.
2. Run docs locally with `npm install` then `npm run dev`.
3. Keep changes small and focused.
4. Run relevant tests for code changes.
5. Open a PR with summary and validation steps.
