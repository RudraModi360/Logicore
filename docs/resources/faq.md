## Do I need a specific model provider?
No. Logicore supports multiple providers with a consistent agent API.

## How do I run docs locally?
Use `npm install` and then `npm run dev`.

## Where do I set API keys?
Use environment variables for each provider and never hardcode secrets.

## Does Logicore support streaming?
Yes. Use streaming mode in agent chat workflows when supported by the provider.
