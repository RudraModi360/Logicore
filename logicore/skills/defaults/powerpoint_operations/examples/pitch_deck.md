# Example: Professional Pitch Deck

## Input
"Create a 5-slide startup pitch deck about AI Agents"

## Strategy
Generate Python code (complex design needed).

## Approach
Write complete python-pptx script with:
- Custom color theme (primary: #2F5496, accent: #4472C4)
- Title slide with company name
- Problem slide with bullet points
- Solution slide with key features
- Market slide with data
- Team slide with roles

## Key Decisions
- Used python-pptx (not skill tool) because need custom colors, fonts, layouts
- Each slide has one key message
- Consistent typography: titles 36pt bold, body 20pt regular
- Speaker notes added for each slide

## Expected Output
A 5-slide presentation with professional styling, custom colors, and speaker notes.
