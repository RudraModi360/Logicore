# Example: Project Report Document

## Input
"Create a project status report with sections for Progress, Risks, and Next Steps"

## Strategy
Generate Python code (structured document with headings and tables).

## Approach
Write python-docx script with:
- Title page with project name and date
- Heading 1 sections: Progress, Risks, Next Steps
- Tables for detailed data
- Consistent formatting throughout

## Key Decisions
- Used python-docx (not skill tool) because need heading hierarchy and tables
- Font: Calibri 11pt body, 18pt headings
- Tables with header row styling

## Expected Output
A multi-page Word document with proper structure, headings, and tables.
